# What is the Games Library?
The Games Library is an application that allows you to save and catalog your video game collection.

## How does it work?

### 1. Create an account using an email and password or log in if you already have one.

![image](https://github.com/JarrodBogard/game-library-frontend/assets/98401586/cc4f9f04-826e-4033-ae38-780f18388438)

### 2. View your catalog of games from the homescreen.

![image](https://github.com/JarrodBogard/game-library-frontend/assets/98401586/e264ecb1-eb77-4c4a-a266-06752104f458)

### 3. Add new games to the catalog by using the input fields at the bottom of the homepage. Add the title, price, and release date, and then click the "Add Game" button.  

![image](https://github.com/JarrodBogard/game-library-frontend/assets/98401586/7ae78f85-440b-4824-8b96-7b0d74bac92d)

![image](https://github.com/JarrodBogard/game-library-frontend/assets/98401586/33447583-361b-4d6f-86c7-fdc30a888f1b)

### 4. To add an image to the game card, choose an image file from your local drive and upload the image.

![image](https://github.com/JarrodBogard/game-library-frontend/assets/98401586/5b95cbc3-820f-4525-b83b-1b17fa66092d)

![image](https://github.com/JarrodBogard/game-library-frontend/assets/98401586/1cb8786c-a1ee-4263-baf7-2a5b11f3be14)

### 5. To delete a game, click on the trash can icon in the top right corner of the game card.
![image](https://github.com/JarrodBogard/game-library-frontend/assets/98401586/aae4873c-1185-457e-849b-82a5e9954db9)

![image](https://github.com/JarrodBogard/game-library-frontend/assets/98401586/6c528bde-a8bb-4a24-ae0c-0c3f0a829fe3)
